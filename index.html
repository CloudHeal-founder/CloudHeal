🚀 Get Started in Seconds

Clone the repo and run your first scan:

# Install dependencies
pip install boto3 google-cloud-storage azure-storage-blob oci networkx flask fpdf2 tabulate
# Scan your current AWS account with auto‑fix
python scanner.py 127.0.0.1 --cloud --fix --db --yes
# Launch the web dashboard
python scanner.py 127.0.0.1 --dashboard
# Scan multiple accounts across all clouds
python scanner.py 127.0.0.1 --chain --accounts aws:123456789012,gcp:my-project,azure:sub-123,oci:comp-456
